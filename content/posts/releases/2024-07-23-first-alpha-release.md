## First Alpha Release

<h2>What's Changed</h2>
<p>This is the first alpha release for <a href="https://nicolino.ralsina.me" rel="nofollow">Nicolino</a>, a fast-ish static site generator written in Crystal.</p>
<ul>
<li>Jinja-like templates courtesy of <a href="https://straight-shoota.github.io/crinja/" rel="nofollow">Crinja</a></li>
<li>Hugo-like shortcodes from <a href="https://github.com/ralsina/shortcode">shortcode</a></li>
<li>Incremental builds using <a href="https://github.com/ralsina/croupier">Croupier</a></li>
<li>Markdown rendering using <a href="https://www.pell.portland.or.us/~orc/Code/discount/" rel="nofollow">Discount</a></li>
</ul>
<p>And <em>some</em> more :-)</p>
